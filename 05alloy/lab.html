Lab - Alloy

Mission: To create an app and it's UI structure using Alloy.

Specification: Using the graphics provided, you will create a new Alloy app. Use the 2-tab template provided by Studio. Then, add views and styles and update the contents of the index files to match the features listed below. Test your work in the simulator or emulator.

To successfully complete this lab, you must develop an application that meets the following description:

  • The app has two tabs, Fugitives and Captured, with icons as shown in the screenshot.
  • You will set a custom background graphic and title bar color (on iOS).
  • Each tab will contain an empty table as shown.
1. Create a new Alloy-based project using the 2-tab template. Use a name and app ID of your choice, or use 05_alloy as the app name and com.yourname.tcd05alloy as the app ID. Studio will open the Resources/app.js and tiapp.xml files for you; close both of them.
The name portion of an app IDs must start with a letter, so com.yourname.05alloy isn't a valid app ID.
2. Download the graphic assets for this application from S3: 05_alloy.zip. Unzip them to the app/assets folder.
3. Update the index.xml and create new View files as necessary to define the two-tabbed app structure. Each tab must contain a window. The tab and window titles should match (use "Fugitives" and "Captured"). Set the appropriate tab icons. Bonus points for using Require and separate files for the two tab's contents.
Tip: Right-click a folder within 'app' and choose 'New Controller' and enter a name (without extension) to create a matched set of View, Style, and Controller files.
4. Update the index.tss and create new Styles files as necessary to set the background graphic and bar color (for iOS). Set the table's foreground color to white. We won't be adding any rows in this lab. But that will set divider colors on iOS.
5. Build your app for the iOS simulator or Android emulator. Compare your results to the screenshot. Correct any differences that exist.

Summary

In this lab, you created an Alloy-based two-tabbed app with custom background graphics and colors.
